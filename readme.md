### Development Setup
- Firstly, setup a virtual environment:  
`python -m venv venv`  

- Activate it:  
 Windows: `.\venv\Scripts\Activate.ps1`
- Packages setup:  
To install the python package dependencies of this project:  
`pip install -r requirements.txt`


### Running
- Run `main.py`:  
`python .\for_the_server\revised_code\main.py`
